## The Open-Source Revolution in Life Science

This repository contains the source materials for the presentation **The Open-Source Revolution in Life Sciences** by Eric Nantz at the 2025 Grand Valley State University [Statistics Career Day](https://www.gvsu.edu/stat/statistics-career-day-2025-123.htm) event on November 17, 2025.

This presentation was authored with the [Quarto](https://quarto.org) scientific publishing system and managed with the [Nix](https://nixos.org/) package management framework assisted with the [`{rix}`](https://docs.ropensci.org/rix/) R package.
